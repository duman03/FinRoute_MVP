# Secrets and Accounts

Fill these values outside the repo before submission.

## App Store Connect

- Apple ID
- App Store Connect app ID
- Apple Team ID
- Bundle identifier confirmation
- App privacy answers

## Google Play Console

- Play Console service account or publisher access
- Package name confirmation
- Data safety answers
- Content rating answers

## Push notifications

- EAS project ID linked to the production app
- `EXPO_PUSH_ACCESS_TOKEN` if push access tokens are enforced
- `mobile/google-services.json`
- `mobile/GoogleService-Info.plist`
- Expo Push Service production delivery verification

## Domains

- `finrouteapp.com`
- `api.finrouteapp.com`
- TLS certificates for both domains
