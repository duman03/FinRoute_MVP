# Screenshot Plan

Capture at least the following screens for both iOS and Android store listings:

1. Login screen
2. Portfolio overview with holdings
3. Trade modal open
4. Gamification screen
5. Leaderboard screen
6. Settings screen with privacy/account controls

Recommended capture rules:

- use a clean seeded account
- avoid placeholder or broken data
- keep notification prompts out of the screenshots
- use consistent language across all screenshots
- show that the app uses simulated trading, not real money
